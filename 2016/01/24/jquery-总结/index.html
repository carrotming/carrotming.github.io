jquery学习总结

jquery
  • jquery

1.添加结点

  • 内部追加

    • .prepend(); 在ul之后添加标签

    • .prependTo(); 在第一个li之前添加标签

    • .append();把newtag添加到第一个li之前

    • .appendTo();在指定元素之后追加

      $('ul').prepend(newtag);
$(newtag).prependTo('li');
$('ul').append(newtag);
$(newtag).appendTo('li');

  • 外部追加

    • .after();
    • .before();
    • .insertBefore();insertBefore() 方法在被选元素之前插入 HTML 标记或已有的元素。

    • .insertAfter();

      $('ul').after(newtag);
$('ul').before(newtag);
$(newtag).insertBefore('ul');
$(newtag).insertAfter('ul');

2.删除结点

  • .remove(); 把当前结点删除

  • .empty(); 清空结点当中的所有内部.

     把当前结点删除
$('ul').remove();
清空结点当中的所有内部.
$('ul').empty();

3.替换结点

  • .replaceWith()

    把p结点替换成newTag
$('p').replaceWith(newTag);

    4.复制结点

  • .clone()

    $('button').click(function () {
    复制第一个li
   var newTag = $('li:first').clone();
   $('ul').append(newTag);

});

事件冒泡
js当中的,冒泡需要兼容w3c和IE的,我们要做一个兼容的封装函数,再进行调用.jq已经将兼容的功能放到了后台.我们只需要调用就可以了.
冒泡行为, 在页面当中重叠了多个元素之后, 都绑定了同一个事件.就会出现冒泡问题.
阻止事件冒泡
事件的默认行为:
on 可以绑定多个事件.
